# Blades and Spells

Small demo. Made just for fun using really outdated stuff.

## Tech Stack

* C99
* OpenGL 2.1
* GLUT
* OpenAL

## Setup

Project uses macOS built-in libraries and frameworks and XCode 14.

## Build

Open project in XCode, _Product_ -> _Build_.
